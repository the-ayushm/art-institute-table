# Art Institute Artwork Table

This project displays artworks from the Art Institute of Chicago API.

Features implemented:
- Server-side pagination using API page parameter
- Row selection with checkboxes
- Persistent selection across pages using stored row IDs
- Custom panel to select N rows across all pages

Tech stack:
- React + TypeScript
- Vite

API used:
https://api.artic.edu/api/v1/artworks?page=1
